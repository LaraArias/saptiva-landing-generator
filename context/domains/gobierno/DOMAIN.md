# Gobierno Domain - Bounded Context

## Ubiquitous Language
Estos términos tienen significado específico para este segmento:

| Término | Significado en Gobierno |
|---------|-------------------------|
| Control | Jurisdicción, soberanía, independencia |
| Seguridad | Requisito legal, no negociable |
| Compliance | CLOUD Act, residencia de datos |
| Velocidad | Secundario al control |

## Audiencia Primaria
- Director de TI de Dependencia
- Responsable de Transformación Digital
- Oficial de Seguridad de Información

## Driver Principal
**Soberanía + Compliance**

No adoptan IA porque:
1. No pueden enviar datos a EE.UU.
2. Necesitan control jurisdiccional
3. Auditoría y trazabilidad son obligatorias
