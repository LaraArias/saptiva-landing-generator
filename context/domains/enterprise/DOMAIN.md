# Enterprise Domain - Bounded Context

## Ubiquitous Language
Estos términos tienen significado específico para este segmento:

| Término | Significado en Enterprise |
|---------|---------------------------|
| Valor | Time-to-value, ROI medible |
| Control | Observabilidad, métricas, dashboards |
| Velocidad | 2 semanas a producción |
| Seguridad | Nice-to-have, no blocker |

## Audiencia Primaria
- CTO / VP Engineering
- Director de Transformación Digital
- Head of Data/AI

## Driver Principal
**Velocidad + Eficiencia**

No les importa dónde corren los datos mientras:
1. Sea rápido de implementar
2. Puedan medir el impacto
3. Reduzca costos operativos

## Anti-patterns
- NO enfatizar soberanía de datos (no es su driver)
