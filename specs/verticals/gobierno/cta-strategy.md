# Gobierno CTA Strategy

## Driver del Segmento
**Soberanía + Compliance**

El CTA debe reflejar control y seriedad institucional.

### Texto Recomendado
Opciones (elegir una):
- "Recupera el control de tus datos"
- "Garantiza la soberanía de tu información"
- "Conoce nuestra solución de IA Soberana"

## Anti-patterns para Gobierno

### NO usar:
- "Solicita demo" → trivializa, parece producto de consumo
- "Prueba gratis" → inapropiado para contexto institucional
- "Acelera" / "Rápido" → velocidad no es su driver
- "Empieza hoy" → gobierno no toma decisiones impulsivas
- Referencias a startups o empresas privadas

### SÍ usar:
- Verbos de control: Garantiza, Protege, Asegura, Recupera
- Referencias a soberanía: control, jurisdicción, independencia
- Tono formal: "Conozca", "Solicite", "Agende"
- Referencias a cumplimiento: normativo, legal, auditable

## Consideraciones de Proceso

### Ciclo de venta largo
- El CTA no busca conversión inmediata
- Busca iniciar relación y proveer información
- Whitepaper como lead magnet principal

### Múltiples stakeholders
- El visitante puede no ser el decisor
- CTA debe funcionar para compartir internamente
- Recursos descargables son importantes
